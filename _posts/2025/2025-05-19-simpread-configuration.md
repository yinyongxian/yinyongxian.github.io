# 简阅配置

## 自定义字体
按自己的喜好安装字体，安装字体后可以设置安装的字体。

## 自定义全局样式

隐藏一些不想看到的内容，看起来简洁美观。

```css
sr-rd-desc {
  display: none;
}

sr-rd-footer {
  display: none;
}

.simpread-theme-root {
  background-color: black !important;
}
```

## 导出图片

使用 Capture node screenshot 导出节点 sr-read 为图片，得到一张好看的图片，便于分享。
