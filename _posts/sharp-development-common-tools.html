C# 开发常用工具

工欲善其事，必先利其器。工具用的好，可以很大地提高开发速度。下面一些开发中常用的工具。

IDE 集成开发环境

Visual Studio号称宇宙最强IDE, 功能十分强大，应有尽有，自然是开发的首选。

打包工具 NSIS

NSIS是功能齐全、开源、免费的打包工具。其脚本语言需要花些时间学习，配合NSIS Dialog Designer能设计出精美的安装界面。

自动编译CruiseControl.NET

按时间自动下载最新源代码进行编译，通过邮件发送编译结果。

批处理

批处理是一个十分强大的脚本，可以简化很多重复性的工作。

CruiseControl.NET下载最新代码，Visual Studio进行编译，批处理调用NSIS打包，批处理上传到代码库的安装包位置。一切自动执行，很少有人工参与，大大减少出错的概率。

Windows Performance Toolkit

Windows Performance Toolkit是用来分析性能瓶颈。使用的工具有Windows Performance Recorder和 Windows Performance Analyzer。可查看具体函数的CPU占用率。

.NET Memory Profiler

查看程序中对象的内存大小，进行性能优化和内存溢出排查。

Navicat for MySQL

MySQL数据库工具，操作十分得劲。

Neor Profile SQL

可将程序的MySQL连接先连接到Neor Profile SQL，由Neor Profile SQL再进行MySQL数据库操作，用来查看程序数据库操作，发现数据库瓶颈以进行优化。

Wireshark

数据包分析工具。

Colasoft Packet Builder

数据包发送工具。
