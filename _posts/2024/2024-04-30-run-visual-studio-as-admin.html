管理员启动 Visual Studio

快捷方式

桌面、开始菜单、任务栏、文件管理器等都要都要逐个修改快捷方式。

双击 *.sln 文件

需要修改注册表来实现管理员启动。

Computer\HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\AppCompatFlags\Layers

~ RUNASADMIN

拖入 *.sln 文件到 Visual Studio

需要对文件devenv.exe为用户设置足够的权限。
