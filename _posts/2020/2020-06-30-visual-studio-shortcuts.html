Visual Studio 快捷键

用Visual Studio通常会安装Resharper、AnkhSVN。SubversionSupport for Visual Studio，部分快捷键可能是插件所有。

在Solution Explore中选中当前文件

ReSharper.ReSharper_LocateInSolutionOrAssemblyExplorer，配置为Ctrl+1

在File Explore中选中当前文件

File.OpenContainingFolder，配置为Ctrl+2

Quick Action

把EditorContextMenus.CodeWindow.ExecuteInInteractive (Alt+ Enter (Global))删除。

EditorContextMenus.CodeWindow.QuickActionsForPosition, 配置Ctrl + .

ReSharper_AltEnter, 配置Alt + Enter

将选定行下/上移

Edit.MoveSelectedLinesDown 设置为 Alt+Down Arrow

Edit.MoveSelectedLinesUp 设置为 Alt+Up Arrow

显示 Visual Studio 智能提示灯泡

Visual Studio 安装 ReSharper 插件后，默认不显示 Visual Studio 智能提示灯泡，可以设置同时显示 Visual Studio 和 ReSharper 的智能提示灯泡。
