重命名MySQL数据库

有时候需要重命名数据库，方式如下：

  1. 创建新的数据库
  2. 修改旧数据库表名称
  3. 删除旧数据库

获取所有MySQL数据库表名称

SELECT table_name FROM information_schema.TABLES WHERE TABLE_SCHEMA='database_name';

重命名MySQL数据库

CREATE DATABASE new_database_name;
RENAME TABLE
database_name.table1 TO new_database_name.table1,
database_name.table2 TO new_database_name.table2;
DROP DATABASE db_name;

数据库操作需谨慎：先备份，后操作。
