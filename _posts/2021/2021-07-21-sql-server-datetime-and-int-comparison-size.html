SQL Server datetime 和 int 比较大小

参数化查询时，SQL Server会默认对数据类型进行转换，转成对应的结果后再进行比较。

整数转成日期： 

declare @i int
set @i = 0
SELECT CAST(@i as datetime)

日期转成整数（距1900-01-01的天数）： 

declare @d datetime
set @d = '2021-07-21 00:00:00'
SELECT CAST(@d as int)
