Git 忽略文件

在仓库中打开Git命令行

添加文件 

git update-index --assume-unchanged navigator/std/Data/Config/Global/Properties.xml
git update-index --assume-unchanged navigator/std/Source/TCM.sln

查看文件 

git ls-files -v | grep '^h\ ' | awk '{print $2}'

取消文件 

git update-index --no-assume-unchanged navigator/std/Data/Config/Global/Properties.xml
git update-index --no-assume-unchanged navigator/std/Source/TCM.slnl
