File Finder 文件查找器

File Finder简称FF，是一款软件。在一台Windows电脑上运行后（免安装），让你的电脑变成文件服务器，通过其他终端的浏览器查看并下载文件。

软件功能

  1. 电脑全部文件共享
  2. 日志记录文件访问记录
  3. 可配置开机自动启动

传输文件不需要U盘，也不需要安装客户端。比如可以躺床上，通过手机浏览起下载电脑上的视频观看。

软件界面

极致的简洁:启动即可，无需任何配置。即开即用，即关即止。

网页界面

软件启动后，通过链接查看电脑上的文件，可以配置成公网访问。

可以选择任意的路径，查看文件夹和文件，下载文件。
