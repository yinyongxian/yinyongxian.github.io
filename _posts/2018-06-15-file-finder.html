File Finder 文件查找器

File Finder简称FF，是一个软件。软件功能：在一台Windows电脑上运行后（免安装），让你的电脑变成文件服务器，可通过其他终端的浏览器查看电脑上的文件，支持下载。

软件界面

极致的简洁，启动即可，无需任何配置。即开即用，即关即止。

网页界面

软件运行后，可在浏览器输入 http://localhost:12321 查看电脑上的文件。

  • localhost换成主机IP，可以在局域网内访问。
  • 也可以配置成公网访问。

可以选择任意的路径，查看文件夹和文件，下载文件。
