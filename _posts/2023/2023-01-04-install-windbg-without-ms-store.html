WinDbg Preview

在没有Visual Studio时使用WinDbg进行调试。

可以直接在MS Store上安装。

无法下载时使用命令行安装

  1. Open MS store website and search for Windbg
  2. Copy url and paste it in https://store.rg-adguard.net/ and you will get the link to download appx file
    3. https://apps.microsoft.com/store/detail/windbg-preview/9PGJGD53TN86?hl=zh-cn&gl=cn&rtc=1
  3. 创建目录C:\debug
  4. C:\WINDOWS\system32>Invoke-WebRequest ‘paste url here’ -OutFile "C:\debug\windbg.appx"
    5. Invoke-WebRequest 'http://tlu.dl.delivery.mp.microsoft.com/filestreamingservice/files/4c032be9-8e5e-490e-9d84-3eb4392501ad?P1=1676013430&P2=404&P3=2&P4=VTe6xBI%2f19MIEc0e8iVDgPRcmIljC%2b2ACwt7UMC614nPuRY8CJSY42m9mtW7WOPPwGMbdrc59%2bQqe4QTgshxkA%3d%3d' -OutFile “C:\debug\windbg.appx”
  5. C:\WINDOWS\system32>Add-AppxPackage -Path C:\debug\windbg.appx
  6. 删除C:\debug

可以使用了。
