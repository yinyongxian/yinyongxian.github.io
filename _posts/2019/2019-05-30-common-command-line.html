常用命令行

日常软件开发过程中，使用一些批处理的命令行，可以提高工作效率。

复制代码，存储到以bat为扩张名的文件中即可使用。

查看设备在线状态

Ping常用检测设备是否能正常通信，长Ping也可以发现设备上线。

@echo off
cd /d C:\Users\Administrator
ping 192.168.0.101 -t

替换MySQL数据库

停止mysql服务，替换文件，保留配置文件，重启服务。可以快速替换数据库，提高多版本开发的效率。

net stop mysql

Copy C:\MySQL\my.ini

rd /s /q C:\MySQL
md C:\MySQL

XCOPY D:\MySQL C:\MySQL /s /e /y

Copy my.ini C:\MySQL
del my.ini

net start mysql

pause

删除mysql服务

@echo off

sc delete mysql

exit

查看端口对应程序

@echo off
cd /d C:\Users\Administrator
netstat -ano

pause

清空目录

@echo off

rd D:\

exit

删除启动项

@echo Off

reg delete "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run\xxx" /f

删除注册表

@echo Off

reg delete "HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows\CurrentVersion\App Paths\xxx" /f

运行软件

@echo off

start /WAIT "" "path"

exit
